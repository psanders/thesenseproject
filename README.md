Weather and Wild Life Research Station
====

Hardware

* RPI B+
* PiCamera NoIR(Night Vision)
* Sim900(SainSmart)
* 12v battery pack
* 12v Step Down to 5v
* 12v Solar Panel
* Project Box
* USB Wifi (Optional)
* Weather Sensors(Not Yet)

Software

* Raspbian Wheezy 09-09
* Timelapse script
* Sim900 scripts

Notes: Ensure that you have the camera an serial ports are set to “enable”.
